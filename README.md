# easyScroll

- [x] 直接一个列表，自动添加box，然后滚动
- [x] 速度相关操作, 拖动时间和拖动距离 构成速度
- [x] 惯性滚动
- [x] 触碰停止滚动
- [x] 各种钩子,(refresh)
- [x] 边缘回弹
- [x] 销毁所有事件
- [x] 可配置滚动条
- [x] 配置参数, 不多就几个 
- [x] 水平垂直方向滚动
- [x] 防抖
- [ ] scrollTo 滚动到某位置 scrollBy 相对于当前位置 滚动多少距离 scrollToElement 滚动到某元素
- [ ] 更友好的错误提示信息
- [x] touchend不会触发的问题，目前原因未知，解决办法是主动触发touchend
